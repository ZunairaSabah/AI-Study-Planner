# AI Study Planner

AI Study Planner is a simple application that generates a personalized study timetable based on subject difficulty and exam dates.

## Features
- Priority based study scheduling
- Automatic time allocation
- Simple interface

## Technologies Used
- Python
- Pandas
- NumPy
- Streamlit

## How to Run the Project

1. Install required libraries
pip install -r requirements.txt

2. Run the application
streamlit run app.py
